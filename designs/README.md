# Designs Directory

Place your design images (PNG, JPG, SVG) in the respective part folders:

- `gala/` - Neckline & collar designs
- `front/` - Front panel designs
- `back/` - Back panel designs
- `bazo/` - Sleeve designs
- `shalwar/` - Bottom fabric designs

## Supported Formats
- PNG (recommended for transparency)
- JPG
- SVG

## Naming Convention
Use descriptive names like:
- `vneck.png`
- `floral-embroidery.png`
- `geometric-pattern.jpg`

The API will serve these files at `/designs/{part}/{filename}`
